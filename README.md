# HilbertBench

## What is this?
HilbertBench is a non-intrusive diagnostic framework for evaluating scientific claims in quantum machine learning.

## Design Principles
- Schema-first
- Evidence, not interpretation
- Append-only traces
- Reproducibility-first

## Status
Early-stage (schema design phase)
